\# Burglar Alarm Robot (Software-Based)



This is a software burglar alarm system that detects motion using a webcam.

If motion is detected using frame differencing, it triggers an alarm sound.



\## Features

\- Landing page with background image

\- Live camera view inside GUI

\- Motion detection using frame differencing

\- Siren alert on intrusion

\- Start/Stop monitoring



\## Requirements

Install libraries:

pip install -r requirements.txt



\## Run the Project

python app.py
